# Nes Emulator
Nintendo Entertainment System emulator implementation in C++

# Generating project
```
git clone https://github.com/GeFeC/nes-emulator.git
cd nes-emulator
mkdir build
cd build
cmake .. -G "<GENERATOR>" -DCMAKE_BUILD_TYPE=Release
```
# Usage
```
./nes-emulator rom_name_without_extension
```
